# EduStream

An educational blog platform for sharing resources, lesson plans, and announcements related to education in Ghana. EduStream provides information on university admissions, teaching opportunities, and educational resources for students, teachers, and educators.

## Features

- **Responsive Design** — Seamlessly adapts to mobile, tablet, and desktop screens
- **Dark Mode Support** — Includes dark mode styles for easier reading in low-light environments
- **Teaching Resources** — Organized lesson plans for Kindergarten, Lower Primary, Upper Primary, and JHS
- **University Updates** — Latest admission information from UDS, UCC, and other institutions
- **Job Updates** — GES recruitment announcements and teaching opportunities
- **Recent Posts Sidebar** — Quick access to the latest articles and updates
- **Social Media Integration** — Links to Facebook, WhatsApp, and LinkedIn

## Getting Started

To run this project locally using Visual Studio Code and the Live Server extension:

1. Open the project folder in VS Code
2. Install the **Live Server** extension if not already installed
3. Right-click `index.html` in the Explorer panel
4. Select **Open with Live Server**
5. The project will open in your browser with live reload enabled
